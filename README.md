# Найди пару

Браузерная игра в жанре «Мемори»: открой две карточки и найди одинаковые предметы до окончания таймера.

## Особенности

- 8 уровней с растущим игровым полем
- Обратный таймер с визуальной полосой прогресса
- Умная раскладка карт — одинаковые пары не стоят рядом
- Адаптивное поле: все клетки помещаются на экран
- 24 эстетичные локальные картинки — игра работает офлайн
- Поддержка мобильных устройств

## Быстрый старт

### Вариант 1: открыть файл

Дважды кликни по `index.html` или перетащи его в браузер.

### Вариант 2: локальный сервер

```bash
npm start
```

Открой в браузере: [http://localhost:3000](http://localhost:3000)

> Для локального сервера нужен [Node.js](https://nodejs.org/).

## Как играть

1. Нажми на закрытую карточку, чтобы открыть её.
2. Открой вторую карточку — если предметы совпали, пара остаётся открытой.
3. Если нет — обе карточки снова закрываются.
4. Найди все пары до конца таймера, чтобы пройти уровень.
